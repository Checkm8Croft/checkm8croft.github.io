Level Editors

Tomb Editor (Recommended)

Tomb Editor is a modern and user-friendly level editor for Tomb Raider, offering a wide range of features and tools to create and modify levels with ease.

I highly recommend using Tomb Editor because others are old and deprecated.


Download Here

DxTre3d

DxTre3d is an old Unofficial Tomb Raider level editor that today is deprecated and not user-friendly as Tomb Editor
