Tomb Raider Levels Open Source

A collection of prj2 level files of original levels of classic Tomb Raiders from 1 to 5, including gold versions.

I actually call them "Open Source" beacause are the original levels, modifiable by everyone with Tomb Editor

NOTE

This is a work in progress, this is not a complete project.

In fact, for realizing this project I used a program for extracting the prj files (that are the files containing rooms, enemies, triggers, statics, etc.) from the original game files, but the once extracted, there aren't triggers and moveables, rooms brightness are at the maximum or in some cases at the minimum. So I have to adjust the level files and trying to making them more similar than the original ones. In fact you may notice that some rooms brightness or some texture positions or certain triggers are quite different between the original ones, this is beacuse these prj files are edited by me, I try to do better as I can, but isn't same at 100% of the original ones, so if you want to contribute to this project, you can try to adjust the prj files and make them more similar to the original ones. At the rest, are "Open Source" for a reason.
